Ramificacion en git

Crear ramas

git branch nombre_rama

Ejemplo

git branch rama1

Ver en que rama estamos

git branch

Resultado

*master
rama1

En el ejemplo anterior tenemos 2 ramas, el que tiene el asterisco significa que estamos en parados en esa rama en este caso estamos en la rama master.

Movernos entre ramas y commits

git checkout nombre_rama

Ejemplo

git checkout rama1

Fusionar ramas

Es importante estar posicionado en la rama que va a absorber los cambios de otra, en nuestro caso master y ejecutamos una simple linea de comando

git merge nombre_rama

Ejemplo

git merge rama1

En este momento ya todos los cambios han sido fusionados y podemos enviarlos al repositorio.

Borrrar ramas

git branch -d nombre_rama

Ejemplo

git branch -d rama1
